# ProvaFresa
Produzione placchetta in alluminio

Sviluppo disegno, quotatura, creazione modello 3D, simulazione tramite Fusion360 delle lavorazioni da eseguire con gli utensili presenti in officina

UTENSILI:

Fresa a Spianare Ø20 3 Taglienti 90°,

Punta per Foratura MDI Ø6.8,

Maschiatore Destrorso M8x1.25,

Fresa a candela MDI Ø12 4 Taglienti,

Svasatore MDI 45°,

## ENGLISH

Production of aluminium plate

Development of drawing, dimensioning, creation of 3D model, simulation through Fusion360 of the processes to be performed with the tools present in the workshop

TOOLS:

Flattening cutter Ø20 3 cutting edges 90°,

MDI Drilling Bit Ø6.8,

Right-handed tapper M8x1.25,

Candle End Mill MDI Ø12 4 Cutting Edges,

MDI Countersink 45°,
